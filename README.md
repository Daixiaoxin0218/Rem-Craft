# Rem Craft

一款思源笔记软件的自用主题。

——的**草稿**。

## 前排提示
由于是自用，因此有部分对其他用户来说可能过于激进的改进：
- 文档树上父子文档的图标对齐（可凭左边框的位置来判断）
- 设置等大宽度菜单的关闭键被隐藏（点击菜单四周的空白处即可关闭）
- 超链接的锚文本编辑功能被隐藏（可直接在编辑器内编辑）
